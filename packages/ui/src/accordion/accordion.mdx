# Accordion 컴포넌트

Accordion 컴포넌트는 콘텐츠를 접을 수 있는 섹션으로 구성하는 UI 컴포넌트입니다.
사용자가 헤더를 클릭하여 해당 섹션의 콘텐츠를 펼치거나 접을 수 있으며,
여러 섹션을 동시에 열어둘 수 있습니다.

## 기본 사용법

```tsx
import { Accordion } from '@/components/accordion';

<Accordion.Provider>
  <Accordion.Button id="1">
    <span>Section Title</span>
    <Accordion.Icon id="1" size={16} />
  </Accordion.Button>

  <Accordion.Panel id="1">
    <div>Section content goes here</div>
  </Accordion.Panel>
</Accordion.Provider>;
```

## 컴포넌트 구조

Accordion은 여러 하위 컴포넌트로 구성된 복합 컴포넌트입니다:

- **Accordion.Provider**: 아코디언 상태를 관리하는 Context Provider
- **Accordion.Button**: 아코디언을 열고 닫는 버튼
- **Accordion.Panel**: 아코디언 콘텐츠를 담는 패널
- **Accordion.Icon**: 열림/닫힘 상태를 나타내는 아이콘

## 사용 예시

### 1. 기본 아코디언

```tsx
<Accordion.Provider>
  <div style={{ maxWidth: '600px' }}>
    <div
      style={{
        border: '1px solid #dee2e6',
        borderRadius: '8px',
        overflow: 'hidden',
      }}
    >
      <Accordion.Button
        id="1"
        etcStyles={{
          width: '100%',
          padding: '16px 20px',
          backgroundColor: '#f8f9fa',
          border: 'none',
          borderBottom: '1px solid #dee2e6',
          cursor: 'pointer',
          display: 'flex',
          justifyContent: 'space-between',
          alignItems: 'center',
          fontSize: '16px',
          fontWeight: '600',
          color: '#495057',
          '&:hover': {
            backgroundColor: '#e9ecef',
          },
        }}
      >
        <span>What is React?</span>
        <Accordion.Icon id="1" size={16} color="#6c757d" />
      </Accordion.Button>

      <Accordion.Panel id="1">
        <div
          style={{
            padding: '20px',
            backgroundColor: 'white',
          }}
        >
          <p style={{ margin: 0, lineHeight: '1.6', color: '#495057' }}>
            React is a JavaScript library for building user interfaces.
          </p>
        </div>
      </Accordion.Panel>
    </div>
  </div>
</Accordion.Provider>
```
