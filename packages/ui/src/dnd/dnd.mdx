# DnD 컴포넌트

DnD(Drag and Drop) 컴포넌트는 사용자가 요소를 드래그하여 다른 위치로 이동시킬 수 있는 인터랙티브한 UI를 제공합니다.
파일 관리자, 칸반 보드, 정렬 가능한 리스트 등 다양한 용도로 사용할 수 있습니다.

## 기본 사용법

```tsx
<DnD.Provider>
  <DnD.Boundary width="100%" height="400px">
    {/* 드래그 가능한 요소들 */}
    <DnD.SingleDraggable
      dragAction={() => console.log('드래그 시작')}
      etcStyles={{
        padding: '12px',
        backgroundColor: '#007bff',
        color: 'white',
        borderRadius: '8px',
        cursor: 'grab',
      }}
    >
      {({ isDrag }) => (
        <div style={{ opacity: isDrag ? 0.5 : 1 }}>드래그 가능한 아이템</div>
      )}
    </DnD.SingleDraggable>

    {/* 드롭 영역 */}
    <DnD.Droppable
      dropAction={(e) => {
        e.preventDefault();
        console.log('드롭됨!');
      }}
      etcStyles={{
        minHeight: '100px',
        border: '2px dashed #6c757d',
        borderRadius: '8px',
        display: 'flex',
        alignItems: 'center',
        justifyContent: 'center',
      }}
    >
      {({ isDragEnter }) => (
        <div style={{ color: isDragEnter ? '#007bff' : '#6c757d' }}>
          {isDragEnter ? '드롭하세요!' : '드롭 영역'}
        </div>
      )}
    </DnD.Droppable>

    {/* 커스텀 포인터 콘텐츠 */}
    <DnD.PointerContent />
  </DnD.Boundary>
</DnD.Provider>
```
