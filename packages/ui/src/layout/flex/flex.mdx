# Flex 컴포넌트

Flex 컴포넌트는 CSS flex-box 쉽게 사용할 수 있도록 도와주는 레이아웃 컴포넌트입니다.

다양한 정렬 옵션과 간격 설정을 제공하며, Polymorphic 컴포넌트로 다양한 HTML 요소로 렌더링할 수 있습니다.

## 기본 사용법

```tsx
import Flex from '@/layout/flex';

// 기본 사용법 - div로 렌더링
<Flex>
  <div>Flex Item 1</div>
  <div>Flex Item 2</div>
</Flex>;
```

## 사용 예시

### 1. 기본 가로 정렬

```tsx
<Flex direction="row" justify="space-between" align="center">
  <div>Left</div>
  <div>Center</div>
  <div>Right</div>
</Flex>
```

### 2. 세로 정렬

```tsx
<Flex direction="column" justify="center" align="center" gap="10px">
  <h2>Title</h2>
  <p>Description</p>
  <button>Action</button>
</Flex>
```

### 3. 간격 설정

```tsx
<Flex direction="row" gap="20px">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</Flex>
```

### 4. 커스텀 스타일 적용

```tsx
<Flex
  direction="column"
  justify="center"
  align="center"
  etcStyles={{
    minHeight: '100vh',
    background: 'linear-gradient(135deg, #667eea 0%, #764ba2 100%)',
    padding: '20px',
  }}
>
  <h1 style={{ color: 'white' }}>Centered Content</h1>
  <p style={{ color: 'white' }}>With custom background</p>
</Flex>
```

### 5. 다양한 HTML 요소로 렌더링

```tsx
// nav 요소로 렌더링
<Flex as="nav" direction="row" justify="space-between" align="center">
  <div>Logo</div>
  <div>Menu Items</div>
</Flex>

// main 요소로 렌더링
<Flex as="main" direction="column" gap="20px">
  <section>Content Section 1</section>
  <section>Content Section 2</section>
</Flex>
```
