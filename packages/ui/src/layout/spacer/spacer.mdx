# Spacer 컴포넌트

Spacer 컴포넌트는 요소들 사이에 공간을 만드는 레이아웃 컴포넌트입니다.
수직 또는 수평 방향으로 지정된 크기의 공간을 생성하여 레이아웃을 구성하는 데 도움을 줍니다.

## 기본 사용법

```tsx
import Spacer from '@/layout/spacer';

// 수직 공간 생성
<Spacer direction="vertical" space="20px" />

// 수평 공간 생성
<Spacer direction="horizontal" space="30px" />
```

## 사용 예시

### 1. 수직 공간 생성

```tsx
<div style={{ display: 'flex', flexDirection: 'column' }}>
  <div>첫 번째 요소</div>
  <Spacer direction="vertical" space="20px" />
  <div>두 번째 요소</div>
</div>
```

### 2. 수평 공간 생성

```tsx
<div style={{ display: 'flex', alignItems: 'center' }}>
  <div>왼쪽 요소</div>
  <Spacer direction="horizontal" space="30px" />
  <div>오른쪽 요소</div>
</div>
```

## 사용 시나리오

### 1. 카드 레이아웃

```tsx
<div style={{ display: 'flex', flexDirection: 'column' }}>
  <h3>카드 제목</h3>
  <Spacer direction="vertical" space="10px" />
  <p>카드 내용</p>
  <Spacer direction="vertical" space="15px" />
  <button>액션 버튼</button>
</div>
```

### 2. 네비게이션 바

```tsx
<nav style={{ display: 'flex', alignItems: 'center' }}>
  <div>로고</div>
  <Spacer direction="horizontal" space="auto" />
  <div>메뉴 1</div>
  <Spacer direction="horizontal" space="20px" />
  <div>메뉴 2</div>
</nav>
```
